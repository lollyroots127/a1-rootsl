Web Dev Concepts

This file is named index.html, and it is the designated homepage of the rootsl/a1-rootsl directory located on the OSU Engineering web server. When a web browser sends an HTTP request for the directory, the Apache web server, the software used by OSU Engineering, maps the request to this file. The majority of web servers are configured to search for an index.html file whenever they receive a request for /, the root directory. On some web servers, other file names are accepted as homepages. For example, Microsoft web servers will also look for default.html files.
